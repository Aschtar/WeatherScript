# Weather XML explained:

- Weather = Self explanatory
- Time = When the desired Weather should start
- Persistent = If the Weather is Persistent and not gets Overwritten by GTA's Weather

- The Weather types:

ExtraSunny,
Clear,
Clouds,
Smog,
Foggy,
Overcast,
Raining,
ThunderStorm,
Clearing,
Neutral,
Snowing,
Blizzard,
Snowlight,
Christmas
